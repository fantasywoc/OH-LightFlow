## 概述

LightFlowMap 项目维护了两种不同的位置标记：
- **图片位置标记**：从照片中提取 GPS 信息生成，带有图片图标
- **搜索/长按位置标记**：通过地址搜索或地图长按生成，使用系统定位图标

两种标记各自独立管理，但共享相同的事件处理系统。

## 数据结构

### PicInfo 接口
```typescript
export interface PicInfo {
  longitude: number;    // 经度
  latitude: number;     // 纬度
  uri: string;          // 图片URI（图片标记有值，搜索标记为空）
  address: string;      // 地址描述
  marker: map.Marker | null;  // 地图标记对象
}
```

### 主要变量

| 变量名 | 类型 | 说明 |
|--------|------|------|
| `picMap` | `Map<string, PicInfo>` | 所有标记的数据存储（key: markerId） |
| `picSet` | `Set<string>` | 图片标记集合（通过URI去重） |
| `searchMarkerId` | `string \| null` | 搜索/长按标记ID（仅一个） |
| `locMap` | `Map<string, number>` | 同一位置标记计数 |
| `lastClickMarker` | `map.Marker \| null` | 当前选中的标记 |
| `address` | `string` | 当前显示的地址 |

export async function getImagePosition(context: Context, uri: string): Promise<PicInfo> {
  // 读取图片EXIF数据
  let imageSource = getImageSource(uri);
  let key = [GPS_LONGITUDE, GPS_LONGITUDE_REF, GPS_LATITUDE, GPS_LATITUDE_REF];
  let data = await imageSource.getImageProperties(key);
  
  // 解析GPS坐标（度分秒转小数）
  // 度 + 分/60 + 秒/3600
  let loStr = data.GPSLongitude.split(',');
  let longiVal = parseInt(loStr[0]) + parseInt(loStr[1])/60 + parseInt(loStr[2])/3600;
  
  // 逆向地理编码获取地址
  let result = await site.reverseGeocode(context, params);
  return { longitude, latitude, uri, address: result.addressDescription, marker: null };
}
```

private async addSearchMarker(locInfo: PicInfo) {
  // 先删除旧的搜索标记（确保只有一个）
  if (this.searchMarkerId) {
    const oldMarker = this.getMarkerById(this.searchMarkerId);
    if (oldMarker) oldMarker.remove();
    this.picMap.delete(this.searchMarkerId);
  }
  
  // 添加新标记
  const markerOptions: mapCommon.MarkerOptions = {
    position: { latitude: locInfo.latitude, longitude: locInfo.longitude },
    icon: $r('sys.symbol.local'),  // 使用系统定位图标
    clickable: true
  };
  const marker = await this.mapController.addMarker(markerOptions);
  
  // 存储
  locInfo.marker = marker;
  this.picMap.set(marker.getId(), locInfo);
  this.searchMarkerId = marker.getId();  // 保存搜索标记ID
}
```

```typescript
private async handleLongPress() {
  // 1. 检查是否有图片标记被选中
  if (this.lastClickMarker) {
    const picInfo = this.picMap.get(markerId);
    if (picInfo && picInfo.uri) {
      await this.updateMarkerPosition(this.lastClickMarker, this.longPressLatLng, picInfo);
      return;
    }
  }
  
  // 2. 检查是否有搜索标记被选中
  if (this.lastClickMarker && this.searchMarkerId) {
    if (markerId === this.searchMarkerId) {
      await this.updateMarkerPosition(this.lastClickMarker, this.longPressLatLng, picInfo);
      return;
    }
  }
  
  // 3. 搜索标记存在但没被选中
  if (this.searchMarkerId) {
    showToast('没有位置被选中');
    return;
  }
  
  // 4. 没有搜索标记，添加新标记
  await this.addLongPressMarker(this.longPressLatLng);
}
```

private async addLongPressMarker(latLng: mapCommon.LatLng) {

## 点击事件响应机制

### 标记点击监听 (第97-114行)

```typescript
let markerCallback = (marker: map.Marker) => {
  const markerId = marker.getId();
  const picInfo = this.picMap.get(markerId);
  
  if (picInfo) {
    if (this.lastClickMarker === marker) {
      // 再次点击同一标记：隐藏地址
      this.address = '';
      this.lastClickMarker = null;
    } else {
      // 点击新标记：显示地址
      this.address = picInfo.address;
      this.lastClickMarker = marker;
    }
  }
};
this.mapEventManager.on('markerClick', markerCallback);
```

### 点击效果

| 场景 | 行为 |
|------|------|
| 第一次点击标记 | 显示地址，标记被选中 |
| 再次点击同一标记 | 隐藏地址，取消选中 |
| 点击不同标记 | 切换显示新标记的地址 |

## 位置更新逻辑

### 长按更新流程 (第494-555行)

#### 优先级：
1. **图片标记被选中** → 更新该图片标记
2. **搜索/长按标记被选中** → 更新该搜索标记
3. **搜索/长按标记存在但未选中** → 提示"没有位置被选中"
4. **无搜索/长按标记** → 在长按位置创建新标记

#### 更新标记位置 (第533-555行)
```typescript
private async updateMarkerPosition(marker: map.Marker, latLng: mapCommon.LatLng, picInfo: PicInfo) {
  // 更新地图上的标记位置
  await marker.setPosition(latLng);
  
  // 更新数据
  picInfo.latitude = latLng.latitude;
  picInfo.longitude = latLng.longitude;
  
  // 更新地址信息
  picInfo.address = await getAddressFromCoordinates(latLng.latitude, latLng.longitude);
  if (this.lastClickMarker === marker) {
    this.address = picInfo.address;
  }
  
  showToast('标记位置已更新', 2000);
}
```

private getMarkerById(markerId: string): map.Marker | null {
  let entries = this.picMap.entries();
  for (let entry of entries) {
    let id = entry[0];
    let info = entry[1];
    if (id === markerId && info.marker) {
      return info.marker;
    }
  }
  return null;
}
